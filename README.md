# ADBTool

一款 Android 原生 ADB 调试工具，可通过 WiFi 网络连接并管理 Android 设备。无需电脑，直接在手机/平板上对另一台 Android 设备进行 ADB 操作。

## 功能特性

### 🔗 设备连接
- 通过 IP 地址和端口连接目标设备（默认 5555 端口）
- 局域网自动扫描，发现开启 ADB 网络调试的设备
- 连接状态管理，记住上次连接的设备

### 📱 应用管理
- 查看已安装应用列表（可切换显示/隐藏系统应用）
- 禁用/启用应用
- 卸载应用（支持卸载预装应用）
- 批量卸载选中应用
- 清除应用数据

### 📁 文件管理
- 浏览设备文件系统
- 目录导航（前进/后退/回到主目录）
- 显示文件详细信息（大小、日期、权限）

### 💻 终端命令
- 直接执行 ADB Shell 命令
- 8 个常用快捷命令按钮（查看应用、设备信息、电池状态等）
- 命令历史记录，支持快速重复执行

## 技术栈

- **语言**: Kotlin
- **最低版本**: Android 7.0 (API 24)
- **目标版本**: Android 14 (API 34)
- **架构**: Fragment + ViewPager2 + BottomNavigation
- **ADB 通信**: 基于 Socket 实现 ADB 协议客户端
- **异步处理**: Kotlin Coroutines

## 构建

```bash
# 调试版本
./gradlew assembleDebug

# 发布版本
./gradlew assembleRelease
```

APK 输出路径: `app/build/outputs/apk/`

## 使用说明

1. 目标设备需先开启「开发者选项」→「USB 调试」→「网络 ADB 调试」
2. 确保两台设备在同一局域网
3. 打开 ADBTool，输入目标设备 IP 或点击「扫描」自动发现
4. 连接成功后即可管理应用、浏览文件、执行命令

## 注意事项

- 某些操作（如卸载系统应用）不可逆，请谨慎操作
- 目标设备需要开启网络 ADB 调试功能
- 部分厂商 ROM 可能限制某些 ADB 命令的执行
